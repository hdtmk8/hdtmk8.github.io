Set与WeakSet

hdtmk8 @hdtmk8

Set

用于存储任何类型的唯一值，无论是基本类型还是对象引用。

  • 只能保存值没有键名
  • 严格类型检测，如字符串数字不等于数值型数字
  • 值是唯一的
  • 遍历顺序是添加的顺序，方便保存回调函数

基本使用

使用数组做初始数据：

let hd=new Set(["hdtmk8","alex","make"]);
console.log(hd.values()); //{"hdtmk8","alex","make"}

使用add添加数据，不允许添加重复的值：

let set = new Set();
set.add(1); 
set.add(1)
console.log(set); //Set(1) {1}

Set是严格类型约束的，下面的数值1与字符串1属于两个不同的值：

let set = new Set();
set.add(1); 
set.add("1");
console.log(set); //Set(2) {1,"1"}

与对象做对比：对象属性最终都会转为字符串，即数值1与字符串1属于相同的值：

let obj={1:"hdtmk8","1":"alex"};
console.log(obj); //{1:"alex"}

使用对象做为键名时，会将对象转为字符串后使用：

let obj={1:"hdtmk8","1":alex};
console.table(obj);

let hd={[obj]:"make"};
console.table(hd);

console.log(hd[obj.toString()]);
console.log(hd["[object Object]"])

获取数量

使用size获取元素数量：

let hd=new Set(["hdtmk8","alex"]);
console.log(hd.size); //2

元素检测

使用has检测元素是否存在：

let hd=new Set();
hd.add("hdtmk8");
console.log(hd.has("hdtmk8")); //true

删除元素

使用delete方法删除单个元素，返回值为boolean类型：

let hd=new Set();
hd.add("hdtmk8");
hd.add("alex");

console.log(hd.delete("alex")); //true
console.log(hd.size); //1
console.log(hd,delete("make")); //false 删除不存在的值返回值为false

使用clear删除所有元素

let hd=new Set();
hd.add("hdtmk8");
hd.add("alex");

hd.clear()
console.log(hd.size); //0

数组转换

可以使用点语法或Array.form静态方法将Set类型轩为数组，这样就可以使用数组处理函数了：

const set=new Set(["hdtmk8","alex"]);
console.log([...set]); //["hdtmk8","alex"];
console.log(Array.from(set)); //["hdtmk8","alex"];

移除Set中大于5的数值，使用filter过滤函数：

let hd=new Set("123456789");
hd=new Set([...hd].filter(item=>item<5));
console.log(hd); //{"1","2","3","4"}

去除重复

去除字符串重复：

console.log([...new Set("hdtmk8hdt")].join("")); //hdtmk8

去除数组重复：

const arr=[1,2,3,4,5,1,2];
console.log(...new Set(arr)); //1,2,3,4,5

遍历数据

使用keys()/values()/entries()都可以返回迭代对象，因为set类型只有值所以keys和values方法结果一致：

const hd = new Set(["hdcms", "houdunren"]);
console.log(hd.values()); //SetIterator {"hdcms", "houdunren"}
console.log(hd.keys()); //SetIterator {"hdcms", "houdunren"}

console.log(hd.entries()); //SetIterator {"hdcms" => "hdcms", "houdunren" => "houdunren"}

可以使用forEach遍历Set数据，默认使用values方法创建迭代器。

为了保持和遍历数组参数统一，函数中的value与key是一样的。

let arr = [7, 6, 2, 8, 2, 6];
let set=new Set(arr);
//使用forEach遍历
set.forEach((item,keye)=>console.log(item,key));

也可以使用forof遍历Set数据，默认使用values方法创建迭代器：

//使用for/of遍历
let set = new Set([7, 6, 2, 8, 2, 6]);

for(const value of set){
    console.log(value);
}

交集

获取两个集合中共同存在的元素：

let hd = new Set(['hdtmk8', 'alex']);
let cms = new Set(['make', 'hdtmk8']);


let newSet=new Set(
    [...hd].filter(item=>cms.has(item))
);
console.log(newSet); //{"hdtmk8"}

差集

在集合a中出现但不在集合b中出现元素集合：

let hd = new Set(['hdtmk8', 'alex']);
let cms = new Set(['make', 'hdtmk8']);

let newSet=new Set(
    [...hd].filter(item=>!cms.has(item))
    );
console.log(newSet); //{"alex"}

并集

将两个集合合并成一个新的集合，由于Set特性当然也不会产生重复元素：

let hd = new Set(['hdtmk8', 'alex']);
let cms = new Set(['make', 'hdtmk8']);

let newSet=[...hd,...cms];
console.log(newSet) //{"hdtmk8","alex","make"}

WeakSet

WeakSet结构同样不会存储重复的值，它的成员必须只能是对象类型的值。

  • 垃圾回收不考虑WeakSet，即被WeakSet引用时引用计数器不加一，所以对象不被引用时不管WeakSet是否在使用都将删除
  • 因为WeakSet 是弱引用，由于其他地方操作成员可能会不存在，所以不可以进行forEach( )遍历等操作
  • 也是因为弱引用，WeakSet 结构没有keys( )，values( )，entries( )等方法和size属性
  • 因为是弱引用所以当外部引用删除时，系统自动删除数据时使用 WeakMap

声明定义

以下操作由于数据还是对象类型将产生错误：

new WeakSet(["hdtmk8", "alex"]); //Invalid value used in weak set

new WeakSet("hdtmk8"); //Invalid value used in weak set

WeakSet的值必须为对象类型：

new WeakSet([["hdtmk8"],["alex"]]);

let WeakS=new WeakSet();
Weaks.add(["hdtmk8"]);

将DOM节点保存到WeakSet

let Wset=doucment.querySelectorAll("button").forEach(item=>Wset.add(item));

基本操作

下面是WeakSet常用指令：

const hd=new WeakSet();
const arr=["hdtmk8"];

//添加操作
let num=hd.add(arr);
console.log(num.); //{"hdtmk8"}

//删除操作
hd.delete(arr);

//检索判断
hd.has(arr)

垃圾回收

WeaSet保存的对象不会增加引用计数器，如果一个对象不被引用了会自动删除。

  • 下例中的数组被 arr 引用了，引用计数器+1
  • 数据又添加到了 hd 的WeaSet中，引用计数还是1
  • 当 arr 设置为null时，引用计数-1 此时对象引用为0
  • 当垃圾回收时对象被删除，这时WakeSet也就没有记录了
const hd=new WeakSet();
let arr=["hdtmk8"];
hd.add(arr);
console.log(hd.has(arr));

arr=null
console.log(hd); //WeakSet {Array(1)};

setTimeout(()=>{
    console.log(hd); //WeakSet{}
},1000);

  class Todos {
    constructor() {}
    run() {
      this.items = document.querySelectorAll("ul>li");
      this.lists = new WeakSet();
      this.record();
      this.addEvent();
    }
    addEvent() {
      this.items.forEach(item => {

          //检测WakeSet中是否存在Li元素
